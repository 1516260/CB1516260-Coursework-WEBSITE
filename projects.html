This webpage is for my projects and the current projects I am currently involved with. The projects I am involved with is the Headless Rapsberry Pi Kali Linux Box, My christmas box using ardunio for my project for University.

Raspberry Pi - Kali Linux Box

The project is running Kali linux on the Raspberry pi and make the device headless and able to connect over SSH and VNC.

Equipment List
  • PC
  • Raspberry Pi 2
  • 32gb Micro Sd card
  • Wireless card for Raspberry Pi
  • Kali Linux for Raspberry Pi
  • Keyboard & Mouse
What the Point Of the Device.
The Point of the device created is the Kali Linux box was designed to be accessed on from another device while the Raspberry Pi running Kali only is plugged in to power.

Ardunio (Christmas Machine) Project

The Project is a christmas machine with an arduino base and wil flash Leds, buzz music and read and show the temperature out in a Lcd.

Equipment List
  • Arduino
  • Led - 4
  • resistors - 4
  • Buzzer
  • Temperature Sensor
  • Lcd Screen
  • Button
What the Point Of the Device.
The Point of the device created is when the temperature gets cold the buzzer plays different music depending on the temperature and while this is happeneing there is a Led flashing and a Lcd displaying the output Merry Christmas and the temperature.
